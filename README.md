# 프리미엄 주택 담보 대출 계산기 (can_i_buy_my_house)

DSR 규제를 완벽하게 반영하고, 체증식 상환 로직까지 지원하는 고도화된 주택 담보 대출 계산기입니다.

## 🚀 실행 방법
이 프로젝트는 별도의 서버 설정 없이 브라우저에서 바로 실행 가능한 **정적 웹 애플리케이션**입니다.

1. 이 저장소를 클론하거나 코드를 다운로드합니다.
2. 폴더 내의 `index.html` 파일을 브라우저(Chrome, Safari, Edge 등)로 엽니다.

## ✨ 주요 기능
- **DSR 완벽 반영**: 기존 부채 내역을 추가하여 실제 대출 가능 한도를 정확히 예측합니다.
- **다양한 상환 방식**: 
  - 원리금균등분할상환
  - 원금균등분할상환
  - **체증식 분할상환**: 청년층을 위한 지수형 증가 로직(연 2.5%)이 적용된 정밀 시뮬레이션
- **시각화 차트**: 연차별 원금과 이자 상환 비중을 스택 바 차트로 제공합니다.
- **만원 단위 반올림**: 금융 기관의 안내 방식과 동일하게 깔끔한 금액 표시 정책을 적용했습니다.

## 🛠 기술 스택
- **HTML5 / CSS3**: 글래스모피즘 기반의 현대적인 UI 디자인
- **Vanilla JavaScript**: 외부 라이브러리 의존성을 최소화한 순수 로직
- **Chart.js (CDN)**: 동적 데이터 시각화
- **Google Fonts (CDN)**: Noto Sans KR 폰트 적용

## 📂 파일 구조
- `index.html`: 메인 인터페이스 및 구조
- `script.js`: 대출 엔진 및 차트 제어 로직
- `style.css`: 테마 및 레이아웃 스타일
